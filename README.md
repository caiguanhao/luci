# LuCI

## Build
Create secrets.env file in project's root directory to set API keys.

```
IPGEOLOCATION_API_KEY=xxxxx
```

## Test
To customize host, user or password, create test.config.json file in project's root directory. For example:

```json
{
  "host": "192.168.1.1",
  "user": "user",
  "pass": "password"
}
```
